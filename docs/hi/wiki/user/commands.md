# कमांड रेफरेंस

CraftMenu मेन्यू प्रबंधन के लिए कमांड का एक व्यापक सेट प्रदान करता है।

## बेस कमांड

सभी कमांड `/craftmenu` (उपनाम: `/cm`) का उपयोग करते हैं।

## सामान्य कमांड

### मदद
```
/cm madad [command]
```
सभी कमांड या किसी विशिष्ट कमांड के लिए मदद जानकारी दिखाता है।

### मेन्यू सूची
```
/cm suchi
```
सभी लोड किए गए मेन्यू टेम्पलेट्स की सूची।

### प्लगइन जानकारी
```
/cm jankari
```
प्लगइन संस्करण और आंकड़े दिखाता है।

## मेन्यू कमांड

### मेन्यू खोलें
```
/cm kholo <menu_name> [player]
```
अपने या किसी अन्य खिलाड़ी के लिए मेन्यू खोलता है।

**उदाहरण:**
- `/cm kholo template` - अपने लिए टेम्पलेट मेन्यू खोलें
- `/cm kholo lobby Steve` - खिलाड़ी Steve के लिए लॉबी मेन्यू खोलें

### मेन्यू बंद करें
```
/cm bandkaro [player]
```
अपने या किसी अन्य खिलाड़ी के लिए सक्रिय मेन्यू बंद करता है।

### मेन्यू बनाएं
```
/cm banana <menu_name>
```
आपके वर्तमान स्थान पर एक नया मेन्यू टेम्पलेट बनाता है।

### मेन्यू हटाएं
```
/cm hatao <menu_name>
```
एक मेन्यू टेम्पलेट को हटाता है।

## रिसोर्स पैक कमांड

### रिसोर्स पैक जनरेट करें
```
/cm paket
```
CraftMenu फोल्डर में इमेज और साउंड से रिसोर्स पैक जनरेट करता है।

### इमेज कमांड
```
/cm chitra jaanch
/cm chitra theek [--backup]
/cm chitra resize <image_path> <target_size>
/cm chitra suraksha
/cm chitra bahal <backup_name>
/cm chitra suchi
/cm chitra surakshayen
```
- `jaanch` - ओवरसाइज़्ड इमेज के लिए स्कैन करता है
- `theek` - ओवरसाइज़्ड इमेज को स्वचालित रूप से ऑप्टिमाइज़ करता है
- `resize` - किसी विशिष्ट इमेज को लक्ष्य आकार में रीसाइज़ करता है (16-4096 पिक्सेल)
- `suraksha` - इमेज का बैकअप बनाता है
- `bahal` - बैकअप से इमेज रिस्टोर करता है
- `suchi` - इमेज फोल्डर में सभी इमेज की सूची
- `surakshayen` - सभी उपलब्ध बैकअप की सूची

## कॉन्फ़िगरेशन कमांड

### रीलोड
```
/cm punarlod
```
सभी कॉन्फ़िगरेशन और मेन्यू टेम्पलेट्स को रीलोड करता है।

### भाषा
```
/cm bhasha <lang>
/cm bhasha suchi
```
प्लगइन भाषा बदलता है। अब "lagao" की आवश्यकता नहीं है - बस `/cm bhasha <lang>` का उपयोग करें।

- `/cm bhasha <lang>` - निर्दिष्ट भाषा सेट करता है
- `/cm bhasha suchi` - सभी उपलब्ध भाषाओं की सूची

उपलब्ध भाषाएं:
- `en_US` - अंग्रेजी
- `pt_BR` - पुर्तगाली (ब्राज़ील)
- `es_ES` - स्पेनिश
- `fr_FR` - फ्रेंच
- `de_DE` - जर्मन
- `it_IT` - इतालवी
- `nl_NL` - डच
- `ru_RU` - रूसी
- `pl_PL` - पोलिश
- `tr_TR` - तुर्की
- `uk_UA` - यूक्रेनी
- `ar_SA` - अरबी
- `ja_JP` - जापानी
- `ko_KR` - कोरियाई
- `zh_CN` - चीनी (सरलीकृत)
- `hi_IN` - हिंदी
- `id_ID` - इंडोनेशियाई
- `th_TH` - थाई
- `vi_VN` - वियतनामी

## डीबग कमांड

### डीबग पार्टिकल्स
```
/cm shodhan kan
/cm shodhan kan aakar <value>
```
- `/cm shodhan kan` - सभी डीबग पार्टिकल्स टॉगल करता है (कोलिज़न, कर्सर, सेंटर्स)
- `/cm shodhan kan aakar <value>` - पार्टिकल साइज़ सेट करता है (0.001 से 2.0)

### ग्रिड डीबग
```
/cm shodhan jaali
/cm shodhan jaali sankhya
```
- `/cm shodhan jaali` - ग्रिड विज़ुअलाइज़ेशन टॉगल करता है
- `/cm shodhan jaali sankhya` - ग्रिड नंबर डिस्प्ले टॉगल करता है

### स्वास्थ्य जांच
```
/cm swasthya
```
कंपोनेंट स्वास्थ्य स्थिति दिखाता है।

### पुनर्प्राप्ति
```
/cm punaprapi
```
त्रुटियों से पुनर्प्राप्त करने का प्रयास करता है।

## एडिटर कमांड

मेन्यू और विजेट्स के लिए इन-गेम विज़ुअल एडिटर खोलता है।

### एडिटर खोलें
```
/cm sampadan_soudh
/cm sampadan_soudh <menu_name>
```
- `/cm sampadan_soudh` - एडिटर हब खोलता है
- `/cm sampadan_soudh <menu_name>` - विशेष मेन्यू के लिए एडिटर खोलता है

**आवश्यक अनुमति:** `craftmenu.admin`

## अनुमतियां

| अनुमति | विवरण |
|--------|--------|
| `craftmenu.use` | बेसिक उपयोग (मेन्यू खोलें) |
| `craftmenu.admin` | एडमिन कमांड |
| `craftmenu.open` | मेन्यू खोलें |
| `craftmenu.create` | मेन्यू बनाएं |
| `craftmenu.reload` | प्लगइन रीलोड करें |
| `craftmenu.debug` | डीबग कमांड |
