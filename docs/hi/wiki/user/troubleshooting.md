# समस्या निवारण

CraftMenu के लिए सामान्य समस्याएं और समाधान।

## इमेज दिखाई नहीं दे रहीं

**लक्षण:** इमेज "?" या गायब कैरेक्टर के रूप में दिखती हैं।

**समाधान:**

1. **रिसोर्स पैक पुनः जनरेट करें:**
   ```
   /cm paket
   ```

2. **इमेज स्थान जांचें:**
   - इमेज सबफोल्डर में होनी चाहिए: `plugins/CraftMenu/images/folder/image.png`
   - रूट में नहीं: `plugins/CraftMenu/images/image.png`

3. **इमेज फॉर्मेट सत्यापित करें:**
   - केवल PNG फाइलें समर्थित हैं
   - सही फाइल एक्सटेंशन सुनिश्चित करें (`.png`, न कि `.PNG`)

4. **रिसोर्स पैक लोड होना जांचें:**
   - सर्वर रिसोर्स पैक कॉन्फ़िगर होना चाहिए
   - खिलाड़ी को रिसोर्स पैक स्वीकार करना होगा

5. **प्लगइन रीलोड करें:**
   ```
   /cm punarlod
   ```

## मेन्यू नहीं खुल रहा

**लक्षण:** `/cm kholo` कमांड कुछ नहीं करता।

**समाधान:**

1. **मेन्यू मौजूद है जांचें:**
   ```
   /cm suchi
   ```

2. **कमांड चलाने के बाद कंसोल में त्रुटियाँ जांचें**

3. **YAML सिंटैक्स सत्यापित करें:**
   - YAML वैलिडेटर उपयोग करें
   - गलत इंडेंटेशन जांचें

4. **स्पॉन लोकेशन वैध है सुनिश्चित करें:**
   - वर्ल्ड लोड होना चाहिए
   - लोकेशन एक्सेसिबल होनी चाहिए

## कोलिज़न काम नहीं कर रहा

**लक्षण:** कर्सर विजेट्स डिटेक्ट नहीं करता।

**समाधान:**

1. **डीबग पार्टिकल्स सक्षम करें:**
   ```
   /debugcollision badlo
   ```

2. **कोलिज़न कॉन्फ़िग जांचें:**
   ```yaml
   collision:
     enabled: true
     size: {x: 0.1, y: 0.1, z: 0.1}
   ```

3. **कोलिज़न बॉक्स साइज़ बढ़ाएं** यदि बहुत छोटा है

4. **विजेट पोजीशन जांचें** - कोलिज़न ऑफसेट हो सकता है

## साउंड नहीं बज रहा

**लक्षण:** साउंड एक्शन का कोई प्रभाव नहीं।

**समाधान:**

1. **कस्टम साउंड के लिए:**
   - `.ogg` फाइलें `plugins/CraftMenu/sounds/folder/` में रखें
   - रिसोर्स पैक पुनः जनरेट करें: `/cm paket`

2. **Minecraft साउंड के लिए:**
   - सही फॉर्मेट उपयोग करें: `minecraft:ui.button.click`

3. **एक्शन कॉन्फ़िग में वॉल्यूम सेटिंग्स जांचें**

## प्रदर्शन समस्याएं

**लक्षण:** मेन्यू उपयोग करते समय लैग।

**समाधान:**

1. **इमेज ऑप्टिमाइज़ करें:**
   ```
   /cm chitra jaanch
   /cm chitra theek --backup
   ```

2. **जटिल मेन्यू में एनिमेशन फ्रीक्वेंसी कम करें**

3. **डीबग मोड अक्षम करें:**
   ```yaml
   craftmenu:
     general:
       debug: false
   ```

4. **अपडेट इंटरवल बढ़ाएं:**
   ```yaml
   craftmenu:
     performance:
       update-interval: 2
   ```

## प्लगइन लोड नहीं हो रहा

**लक्षण:** स्टार्टअप पर प्लगइन त्रुटियाँ दिखाता है।

**समाधान:**

1. **Java संस्करण जांचें:**
   - Java 17 या उच्चतर आवश्यक है

2. **डिपेंडेंसीज सत्यापित करें:**
   - PacketEvents इंस्टॉल होना चाहिए

3. **सर्वर संस्करण जांचें:**
   - Minecraft 1.20.4+ आवश्यक है

4. **विशिष्ट त्रुटियों के लिए स्टार्टअप लॉग्स देखें**

5. **पुनर्प्राप्ति प्रयास करें:**
   ```
   /cm punaprapi
   ```

## YAML त्रुटियाँ

**लक्षण:** त्रुटियाँ YAML पार्सिंग का उल्लेख करती हैं।

**सामान्य समस्याएं:**

1. **गलत इंडेंटेशन:**
   ```yaml
   # गलत
   widgets:
   my_widget:
     type: IMAGE

   # सही
   widgets:
     my_widget:
       type: IMAGE
   ```

2. **विशेष वैल्यू के आसपास कोट्स गायब:**
   ```yaml
   # गलत - & का विशेष अर्थ है
   title: &bनमस्ते

   # सही
   title: "&bनमस्ते"
   ```

3. **गलत लिस्ट फॉर्मेट:**
   ```yaml
   # गलत
   events:
     on_cursor_click:
       action: sound

   # सही
   events:
     on_cursor_click:
       - action: sound
   ```

## मदद प्राप्त करना

यदि आपको अभी भी समस्याएं हैं:

1. डीबग मोड सक्षम करें और कंसोल आउटपुट जांचें
2. ज्ञात समस्याओं के लिए GitHub issues देखें
3. निम्नलिखित के साथ नया issue बनाएं:
   - सर्वर संस्करण
   - प्लगइन संस्करण
   - कंसोल लॉग्स
   - कॉन्फ़िगरेशन फाइलें (संवेदनशील डेटा हटाएं)
