# इन-गेम एडिटर

CraftMenu में एक शक्तिशाली विज़ुअल एडिटर शामिल है जो आपको YAML फ़ाइलों को मैन्युअल रूप से संपादित किए बिना, इन्वेंटरी GUI के माध्यम से सीधे मेनू कॉन्फ़िगर करने की अनुमति देता है।

## शुरुआत करना

### एडिटर खोलना

```
/cm sampadan              # मुख्य एडिटर हब खोलें
/cm sampadan <menu>       # किसी विशिष्ट मेनू को सीधे संपादित करें
```

**आवश्यक अनुमति:** `craftmenu.admin` या `craftmenu.edit`

### एडिटर नेविगेशन

एडिटर **स्टैक-आधारित नेविगेशन** सिस्टम का उपयोग करता है:
- **बायाँ-क्लिक** आइटम पर सब-मेनू में प्रवेश करने या मान संपादित करने के लिए
- **दायाँ-क्लिक** द्वितीयक क्रियाओं के लिए (पूर्वावलोकन, परीक्षण)
- **Shift + बायाँ-क्लिक** आइटम हटाने के लिए (पुष्टि के साथ)
- **तीर आइटम** (वापस बटन) पिछले मेनू पर लौटने के लिए
- **इन्वेंटरी बंद करें** या बाहर क्लिक करके बाहर निकलें

---

## एडिटर मुख्य मेनू

जब आप `/cm sampadan` चलाते हैं, आपको इन विकल्पों के साथ मुख्य एडिटर हब दिखाई देगा:

| आइटम | विवरण |
|------|-------------|
| **मेनू सूची** | सभी लोड किए गए मेनू ब्राउज़ और संपादित करें |
| **इमेज ब्राउज़र** | सभी उपलब्ध इमेज देखें |
| **साउंड ब्राउज़र** | सभी उपलब्ध साउंड देखें |
| **कॉन्फ़िग** | ग्लोबल प्लगइन कॉन्फ़िगरेशन |

---

## मेनू संपादन

### मेनू सूची

आपके `menus/` फ़ोल्डर में सभी मेनू दिखाता है। इसका एडिटर खोलने के लिए मेनू पर क्लिक करें।

- **बायाँ-क्लिक**: मेनू संपादित करें
- **Shift + बायाँ-क्लिक**: मेनू हटाएं (पुष्टि के साथ)
- **नया बनाएं**: अपने वर्तमान स्थान पर नया मेनू जोड़ें

### मेनू क्रियाएं हब

मेनू चुनने के बाद, आपको इन अनुभागों के साथ मुख्य मेनू एडिटर दिखाई देगा:

| अनुभाग | विवरण |
|---------|-------------|
| **गुण** | मूल सेटिंग्स (नाम, शीर्षक, मुख्य मेनू, ऑटो-ओपन) |
| **स्थान** | दुनिया में स्थिति और रोटेशन |
| **लेआउट** | ग्रिड लेआउट कॉन्फ़िगरेशन |
| **कीबाइंड** | कीबोर्ड शॉर्टकट |
| **दृश्यता** | प्लेयर/मॉब/आइटम छिपाने की सेटिंग्स |
| **उन्नत** | कर्सर संवेदनशीलता, कैमरा लॉक, सीमाएं |
| **विजेट** | इस मेनू में विजेट संपादित करें |

---

## मेनू गुण

मूल मेनू जानकारी संपादित करें:

| गुण | विवरण |
|----------|-------------|
| **नाम** | मेनू पहचानकर्ता (कमांड में उपयोग) |
| **शीर्षक** | प्रदर्शन शीर्षक (& रंग कोड समर्थित) |
| **विवरण** | वैकल्पिक विवरण |
| **मुख्य मेनू** | प्राथमिक मेनू के रूप में चिह्नित करें |
| **जॉइन पर खोलें** | प्लेयर के सर्वर जॉइन करने पर ऑटो-ओपन |
| **टेलीपोर्ट पर खोलें** | प्लेयर के इस दुनिया में टेलीपोर्ट करने पर ऑटो-ओपन |
| **वर्ल्ड** | वह दुनिया जहां मेनू मौजूद है |

### टेक्स्ट मान संपादित करना

जब आप टेक्स्ट प्रॉपर्टी पर क्लिक करते हैं:
1. इन्वेंटरी बंद हो जाती है
2. चैट में प्रॉम्प्ट दिखाई देता है
3. चैट में अपना नया मान टाइप करें
4. पुष्टि के लिए Enter दबाएं (या रद्द करने के लिए `cancel` टाइप करें)

---

## मेनू स्थान

कॉन्फ़िगर करें कि मेनू दुनिया में कहां दिखाई दे:

| गुण | विवरण |
|----------|-------------|
| **वर्ल्ड** | उपलब्ध दुनियाओं से चुनें |
| **X / Y / Z** | निर्देशांक (चैट के माध्यम से संपादित करने के लिए क्लिक करें) |
| **Yaw** | क्षैतिज रोटेशन (-180 से 180) |
| **Pitch** | लंबवत रोटेशन (-90 से 90) |
| **वर्तमान पर सेट करें** | अपनी वर्तमान स्थिति/रोटेशन का उपयोग करें |

---

## मेनू लेआउट (ग्रिड)

ग्रिड-आधारित विजेट स्थिति कॉन्फ़िगर करें:

| गुण | विवरण |
|----------|-------------|
| **सक्षम** | ग्रिड लेआउट चालू/बंद टॉगल करें |
| **कॉलम** | ग्रिड कॉलम की संख्या |
| **पंक्तियाँ** | ग्रिड पंक्तियों की संख्या |
| **गैप X / Y / Z** | सेल के बीच स्पेसिंग |
| **संरेखण** | ग्रिड संरेखण (CENTER, TOP_LEFT, आदि) |

जब ग्रिड लेआउट सक्षम होता है, विजेट मैन्युअल निर्देशांकों के बजाय `grid-position: {row: X, col: Y}` का उपयोग करते हैं।

---

## मेनू कीबाइंड

कीबोर्ड शॉर्टकट कॉन्फ़िगर करें:

| क्रिया | विवरण |
|--------|-------------|
| **कीबाइंड जोड़ें** | नया कीबोर्ड शॉर्टकट बनाएं |
| **कीबाइंड संपादित करें** | मौजूदा कीबाइंड संशोधित करें |
| **कीबाइंड हटाएं** | कीबाइंड हटाएं |

### कीबाइंड गुण

- **की**: की या कॉम्बिनेशन (जैसे, `SHIFT`, `CTRL+E`, `F`)
- **क्रिया**: `activate`, `toggle`, या `close`
- **विजेट**: लक्ष्य विजेट नाम (activate/toggle के लिए)

---

## मेनू दृश्यता

नियंत्रित करें कि मेनू खुला होने पर क्या दिखाई दे:

| गुण | विवरण |
|----------|-------------|
| **प्लेयर छिपाएं** | अन्य प्लेयर को दृश्य से छिपाएं |
| **मॉब छिपाएं** | सभी मॉब छिपाएं |
| **आइटम छिपाएं** | जमीन पर आइटम छिपाएं |
| **व्हाइटलिस्ट** | जो प्लेयर दिखाई रहें (सूची संपादित करें) |

---

## उन्नत सेटिंग्स

मेनू व्यवहार को फाइन-ट्यून करें:

| गुण | विवरण |
|----------|-------------|
| **कर्सर संवेदनशीलता** | माउस मूवमेंट स्पीड (0.1 - 5.0) |
| **अधिकतम Yaw ऑफसेट** | क्षैतिज कर्सर सीमा (डिग्री) |
| **अधिकतम Pitch ऑफसेट** | लंबवत कर्सर सीमा (डिग्री) |
| **कैमरा लॉक सक्षम** | मेनू खुला होने पर प्लेयर का कैमरा लॉक करें |
| **कैमरा लॉक स्ट्रेंथ** | कैमरा कितनी मजबूती से लॉक है (0.0 - 1.0) |
| **सीमा ध्वनि** | कर्सर के सीमा तक पहुंचने पर ध्वनि |
| **सीमा वॉल्यूम/पिच** | ध्वनि गुण |
| **सीमा संदेश** | सीमा पर दिखाया गया संदेश |

---

## विजेट संपादन

### विजेट सूची

वर्तमान मेनू में सभी विजेट दिखाता है:

- **बायाँ-क्लिक**: विजेट संपादित करें
- **Shift + बायाँ-क्लिक**: विजेट हटाएं
- **नया बनाएं**: नया विजेट जोड़ें

### विजेट एडिटर हब

प्रत्येक विजेट में ये संपादन योग्य अनुभाग हैं:

| अनुभाग | विवरण |
|---------|-------------|
| **प्रकार** | IMAGE, TEXT, या CURSOR |
| **ट्रांसफॉर्म** | स्थिति, आकार, रोटेशन |
| **विज़ुअल स्टेट्स** | सामान्य, होवर, दबाया, अक्षम दिखावट |
| **कोलिज़न** | कोलिज़न बॉक्स कॉन्फ़िगरेशन |
| **इवेंट्स** | इंटरैक्शन इवेंट्स और क्रियाएं |
| **[प्रकार-विशिष्ट]** | विजेट प्रकार के आधार पर अतिरिक्त विकल्प |

---

## ट्रांसफॉर्म एडिटर

विजेट स्थिति और आकार कॉन्फ़िगर करें:

### स्थिति
- **X**: क्षैतिज स्थिति
- **Y**: लंबवत स्थिति
- **Z**: गहराई स्थिति

### आकार
- **X**: चौड़ाई स्केल
- **Y**: ऊंचाई स्केल
- **Z**: गहराई स्केल

### रोटेशन
- **Pitch**: ऊपर/नीचे रोटेशन
- **Yaw**: बाएं/दाएं रोटेशन
- **Roll**: झुकाव रोटेशन

**टिप**: चैट इनपुट के माध्यम से संपादित करने के लिए किसी भी मान पर क्लिक करें।

---

## विज़ुअल स्टेट्स

विभिन्न स्टेट्स के लिए विजेट की अलग-अलग दिखावट हो सकती है:

| स्टेट | कब लागू होता है |
|-------|--------------|
| **normal** | डिफ़ॉल्ट स्टेट |
| **hover** | कर्सर विजेट पर है |
| **pressed** | विजेट क्लिक किया जा रहा है |
| **disabled** | विजेट निष्क्रिय है |
| **कस्टम** | कोई भी कस्टम स्टेट नाम |

### विज़ुअल स्टेट एडिटर

प्रत्येक स्टेट में है:

| गुण | विवरण |
|----------|-------------|
| **प्रकार** | `image`, `text`, या `unicode` |
| **मान** | इमेज पथ, टेक्स्ट सामग्री, या यूनिकोड कैरेक्टर |
| **ओवरराइड्स** | वैकल्पिक ट्रांसफॉर्म/कोलिज़न/टेक्स्ट-साइज़ ओवरराइड्स |

---

## कोलिज़न एडिटर

विजेट के क्लिक करने योग्य क्षेत्र को कॉन्फ़िगर करें:

| गुण | विवरण |
|----------|-------------|
| **सक्षम** | कोलिज़न डिटेक्शन टॉगल करें |
| **स्थिति X/Y/Z** | कोलिज़न बॉक्स सेंटर ऑफसेट |
| **आकार X/Y/Z** | कोलिज़न बॉक्स आयाम |
| **ऑफसेट X/Y/Z** | अतिरिक्त ऑफसेट |

**टिप**: इन-गेम कोलिज़न बॉक्स को विज़ुअलाइज़ करने के लिए `/cm debug particles` का उपयोग करें।

---

## इवेंट्स एडिटर

### इवेंट प्रकार

| इवेंट | ट्रिगर |
|-------|---------|
| **on_menu_open** | जब मेनू खुलता है |
| **on_cursor_hover** | जब कर्सर विजेट में प्रवेश करता है |
| **on_cursor_hover_exit** | जब कर्सर विजेट छोड़ता है |
| **on_cursor_click** | जब विजेट क्लिक किया जाता है |

### एक्शन लिस्ट

प्रत्येक इवेंट में क्रम में निष्पादित होने वाली क्रियाओं की सूची होती है:

- **बायाँ-क्लिक**: एक्शन संपादित करें
- **Shift + बायाँ-क्लिक**: एक्शन हटाएं
- **एक्शन जोड़ें**: नई एक्शन बनाएं
- **पुनर्व्यवस्थित करें**: निष्पादन क्रम बदलने के लिए खींचें

---

## एक्शन एडिटर्स

प्रत्येक एक्शन प्रकार में एक विशेष एडिटर है:

### एनिमेशन एक्शन

| गुण | विवरण |
|----------|-------------|
| **इफेक्ट** | एनिमेशन प्रकार (rotate, scale, bounce, आदि) |
| **अवधि** | मिलीसेकंड में एनिमेशन की लंबाई |
| **स्केल X/Y/Z** | स्केल गुणक (स्केल एनिमेशन के लिए) |
| **तीव्रता** | इफेक्ट स्ट्रेंथ (0.1 - 5.0) |
| **ईज़िंग** | टाइमिंग फंक्शन (linear, ease_in, ease_out, आदि) |
| **प्राथमिकता** | एनिमेशन के दौरान इंटरैक्शन ब्लॉक करें |

### साउंड एक्शन

| गुण | विवरण |
|----------|-------------|
| **फाइल** | साउंड पथ (minecraft:... या कस्टम पथ) |
| **वॉल्यूम** | साउंड वॉल्यूम (0.0 - 1.0) |
| **पिच** | साउंड पिच (0.5 - 2.0) |

**ब्राउज़**: साउंड ब्राउज़र खोलने और साउंड चुनने के लिए क्लिक करें।

### कमांड एक्शन

| गुण | विवरण |
|----------|-------------|
| **कमांड** | निष्पादित करने का कमांड (विशेष कमांड के साथ) |
| **विलंब** | निष्पादन से पहले मिलीसेकंड में विलंब |

**विशेष कमांड:**
- `[TELEPORT] world x y z yaw pitch`
- `[MESSAGE] &aआपका संदेश यहां`
- `[CLOSE]`
- `[PLAY_MUSIC] path/file.ogg`
- `[STOP_MUSIC]`
- `[OPEN_URL] https://...`
- `[PLAYER] /command`
- `[CONSOLE] /command`

### स्टेट एक्शन

| गुण | विवरण |
|----------|-------------|
| **एक्शन प्रकार** | `toggle_state` या `set_state` |
| **स्टेट्स** | टॉगल करने के लिए स्टेट्स की सूची (toggle_state) |
| **स्टेट** | लक्ष्य स्टेट नाम (set_state) |

### विज़ुअल चेंज एक्शन

| गुण | विवरण |
|----------|-------------|
| **To** | लक्ष्य विज़ुअल स्टेट नाम |

### विजेट एक्शन

| गुण | विवरण |
|----------|-------------|
| **एक्शन** | `hide_widget`, `show_widget`, आदि |
| **विजेट** | लक्ष्य विजेट नाम |

### इफेक्ट एक्शन

| गुण | विवरण |
|----------|-------------|
| **इफेक्ट** | लागू करने के लिए इफेक्ट प्रकार |
| **पैरामीटर** | इफेक्ट-विशिष्ट पैरामीटर |

### स्टॉप एनिमेशन एक्शन

| गुण | विवरण |
|----------|-------------|
| **एनिमेशन प्रकार** | कौन सा एनिमेशन रोकना है |

### स्टॉप इफेक्ट एक्शन

| गुण | विवरण |
|----------|-------------|

## एसेट ब्राउज़र्स

### इमेज ब्राउज़र

अपने `images/` फ़ोल्डर में सभी इमेज ब्राउज़ करें:

- **पेजिनेशन**: इमेज के पेजों के माध्यम से नेविगेट करें
- **पूर्वावलोकन**: इमेज पथ और विवरण देखें
- **चयन करें**: वर्तमान संदर्भ में उपयोग करने के लिए क्लिक करें

इमेज फ़ोल्डर द्वारा व्यवस्थित होते हैं (जैसे, `template/button.png`)।

### साउंड ब्राउज़र

अपने `sounds/` फ़ोल्डर में सभी साउंड प्लस Minecraft बिल्ट-इन साउंड ब्राउज़ करें:

- **कस्टम साउंड**: `sounds/` से आपकी .ogg फाइलें
- **Minecraft साउंड**: बिल्ट-इन साउंड (minecraft:ui.button.click, आदि)
- **चयन करें**: वर्तमान संदर्भ में उपयोग करने के लिए क्लिक करें

## टिप्स और सर्वोत्तम प्रथाएं

### वर्कफ़्लो टिप्स

1. **गुणों से शुरू करें**: पहले नाम, शीर्षक और स्थान सेट करें
2. **विजेट जोड़ें**: बुनियादी ट्रांसफॉर्म के साथ अपने विजेट बनाएं
3. **विज़ुअल कॉन्फ़िगर करें**: सामान्य और होवर स्टेट सेट करें
4. **कोलिज़न जोड़ें**: कोलिज़न बॉक्स सक्षम करें और आकार दें
5. **इवेंट्स जोड़ें**: होवर साउंड और क्लिक एक्शन कॉन्फ़िगर करें
6. **बार-बार टेस्ट करें**: परिवर्तनों का परीक्षण करने के लिए `/cm kholo <menu>` का उपयोग करें

### कीबोर्ड शॉर्टकट

| शॉर्टकट | एक्शन |
|----------|--------|
| **Escape** | एडिटर बंद करें |
| **नंबर कीज़ (1-9)** | त्वरित स्लॉट चयन |

### सामान्य समस्याएं

**परिवर्तन दिखाई नहीं दे रहे:**
- परिवर्तन करने के बाद `/cm punahload` चलाएं
- सुनिश्चित करें कि आपने एडिटर में "सेव" पर क्लिक किया

**कोलिज़न का पता नहीं लग रहा:**
- जांचें कि कोलिज़न सक्षम है
- सत्यापित करें कि कोलिज़न आकार पर्याप्त बड़ा है
- विज़ुअलाइज़ करने के लिए `/cm debug particles` का उपयोग करें

**इमेज नहीं दिख रही:**
- रिसोर्स पैक पुनर्जनन करने के लिए `/cm paket` चलाएं
- सुनिश्चित करें कि इमेज एक सबफ़ोल्डर में है (जैसे, `images/mymenu/`)
- क्लाइंट पर रिसोर्स पैक लागू करें
