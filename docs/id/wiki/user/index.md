# Panduan Pengguna CraftMenu

Selamat datang di Panduan Pengguna CraftMenu! CraftMenu adalah sistem menu 3D canggih untuk Minecraft yang memungkinkan Anda membuat menu interaktif dan imersif yang dirender di dalam dunia permainan.

## Apa itu CraftMenu?

CraftMenu mengubah cara pemain berinteraksi dengan menu di Minecraft. Alih-alih menu berbasis inventaris tradisional, CraftMenu merender menu 3D yang indah menggunakan entitas TextDisplay yang melayang di dunia permainan.

## Fitur Utama

- **Rendering Menu 3D** - Menu dirender di dunia permainan menggunakan entitas TextDisplay
- **Interaksi Berbasis Mouse** - Kontrol kursor penuh dengan deteksi hover dan klik
- **Pembuatan Resource Pack Otomatis** - Pembuatan resource pack otomatis untuk gambar dan suara kustom
- **Sistem Animasi** - 19 jenis animasi dengan 6 fungsi easing
- **Dukungan Multi-Bahasa** - Dukungan bawaan untuk 20 bahasa (Inggris, Portugis Brasil, Portugis Portugal, Spanyol, Prancis, Jerman, Italia, Belanda, Rusia, Polandia, Turki, Ukraina, Arab, Jepang, Korea, Cina, Hindi, Indonesia, Thailand, Vietnam)
- **Integrasi PlaceholderAPI** - Tampilkan data pemain dinamis di menu Anda
- **Menu Buka Otomatis** - Buka menu secara otomatis saat pemain bergabung atau teleportasi

## Mulai Cepat

1. **Instal plugin** - Letakkan JAR di folder `plugins/` Anda
2. **Mulai server Anda** - CraftMenu akan membuat konfigurasi default
3. **Buat resource pack** - Jalankan `/cm paket` untuk membuat resource pack
4. **Buka menu** - Gunakan `/cm buka template` untuk menguji menu default
