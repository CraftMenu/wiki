# คู่มือผู้ใช้ CraftMenu

ยินดีต้อนรับสู่คู่มือผู้ใช้ CraftMenu! CraftMenu เป็นระบบเมนู 3D ขั้นสูงสำหรับ Minecraft ที่ช่วยให้คุณสร้างเมนูที่ดื่มด่ำและโต้ตอบได้ซึ่งแสดงผลในโลกเกม

## CraftMenu คืออะไร?

CraftMenu เปลี่ยนวิธีที่ผู้เล่นโต้ตอบกับเมนูใน Minecraft แทนที่จะใช้เมนูแบบ inventory ดั้งเดิม CraftMenu จะแสดงผลเมนู 3D ที่สวยงามโดยใช้ TextDisplay entities ที่ลอยอยู่ในโลกเกม

## คุณสมบัติหลัก

- **การแสดงผลเมนู 3D** - เมนูถูกแสดงผลในโลกเกมโดยใช้ TextDisplay entities
- **การโต้ตอบด้วยเมาส์** - ควบคุมเคอร์เซอร์ได้เต็มรูปแบบพร้อมการตรวจจับการ hover และ click
- **สร้าง Resource Pack อัตโนมัติ** - สร้าง resource pack สำหรับภาพและเสียงที่กำหนดเองโดยอัตโนมัติ
- **ระบบแอนิเมชัน** - รองรับ 19 ประเภทแอนิเมชันพร้อม 6 ฟังก์ชัน easing
- **รองรับหลายภาษา** - รองรับในตัว 20 ภาษา (อังกฤษ, โปรตุเกส (บราซิล), โปรตุเกส (โปรตุเกส), สเปน, ฝรั่งเศส, เยอรมัน, อิตาลี, ดัตช์, รัสเซีย, โปแลนด์, ตุรกี, ยูเครน, อาหรับ, ญี่ปุ่น, เกาหลี, จีน, ฮินดี, อินโดนีเซีย, ไทย, เวียดนาม)
- **เชื่อมต่อ PlaceholderAPI** - แสดงข้อมูลผู้เล่นแบบ dynamic ในเมนูของคุณ
- **เปิดเมนูอัตโนมัติ** - เปิดเมนูอัตโนมัติเมื่อผู้เล่นเข้าร่วมหรือเทเลพอร์ต

## เริ่มต้นอย่างรวดเร็ว

1. **ติดตั้งปลั๊กอิน** - วาง JAR ไว้ในโฟลเดอร์ `plugins/`
2. **เริ่มเซิร์ฟเวอร์** - CraftMenu จะสร้างไฟล์คอนฟิกเริ่มต้น
3. **สร้าง resource pack** - รัน `/cm pakked` เพื่อสร้าง resource pack
4. **เปิดเมนู** - ใช้ `/cm perd template` เพื่อทดสอบเมนูเริ่มต้น
