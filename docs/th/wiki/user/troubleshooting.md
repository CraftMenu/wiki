# การแก้ปัญหา

ปัญหาทั่วไปและวิธีแก้ไขสำหรับ CraftMenu

## ภาพไม่แสดง

**อาการ:** ภาพแสดงเป็น "?" หรืออักขระหายไป

**วิธีแก้ไข:**

1. **สร้าง resource pack ใหม่:**
   ```
   /cm pakked
   ```

2. **ตรวจสอบตำแหน่งภาพ:**
   - ภาพต้องอยู่ในโฟลเดอร์ย่อย: `plugins/CraftMenu/images/folder/image.png`
   - ไม่ใช่ใน root: `plugins/CraftMenu/images/image.png`

3. **ตรวจสอบรูปแบบภาพ:**
   - รองรับเฉพาะไฟล์ PNG
   - ตรวจสอบนามสกุลไฟล์ถูกต้อง (`.png` ไม่ใช่ `.PNG`)

4. **ตรวจสอบว่า resource pack โหลดแล้ว:**
   - ต้องคอนฟิก server resource pack
   - ผู้เล่นต้องยอมรับ resource pack

5. **โหลดปลั๊กอินใหม่:**
   ```
   /cm lodmai
   ```

## เมนูไม่เปิด

**อาการ:** คำสั่ง `/cm perd` ไม่ทำอะไร

**วิธีแก้ไข:**

1. **ตรวจสอบว่าเมนูมีอยู่:**
   ```
   /cm raykan
   ```

2. **ตรวจสอบคอนโซลหาข้อผิดพลาด** หลังจากรันคำสั่ง

3. **ตรวจสอบ syntax ของ YAML:**
   - ใช้ YAML validator
   - ตรวจสอบการย่อหน้าที่ไม่ถูกต้อง

4. **ตรวจสอบตำแหน่ง spawn ที่ถูกต้อง:**
   - โลกต้องถูกโหลด
   - ตำแหน่งต้องเข้าถึงได้

## การชนไม่ทำงาน

**อาการ:** เคอร์เซอร์ไม่ตรวจจับ widget

**วิธีแก้ไข:**

1. **เปิดอนุภาค debug:**
   ```
   /cm kaekhong chon
   ```

2. **ตรวจสอบคอนฟิก collision:**
   ```yaml
   collision:
     enabled: true
     size: {x: 0.1, y: 0.1, z: 0.1}
   ```

3. **เพิ่มขนาดกล่องการชน** ถ้ามันเล็กเกินไป

4. **ตรวจสอบตำแหน่ง widget** - collision อาจถูก offset

## เสียงไม่เล่น

**อาการ:** action เสียงไม่มีผล

**วิธีแก้ไข:**

1. **สำหรับเสียงที่กำหนดเอง:**
   - วางไฟล์ `.ogg` ใน `plugins/CraftMenu/sounds/folder/`
   - สร้าง resource pack ใหม่: `/cm pakked`

2. **สำหรับเสียง Minecraft:**
   - ใช้รูปแบบที่ถูกต้อง: `minecraft:ui.button.click`

3. **ตรวจสอบการตั้งค่า volume** ในคอนฟิก action

## ปัญหาประสิทธิภาพ

**อาการ:** มีความหน่วงเมื่อใช้เมนู

**วิธีแก้ไข:**

1. **ปรับแต่งภาพ:**
   ```
   /cm phap truat
   /cm phap kaekhai --backup
   ```

2. **ลดความถี่ของแอนิเมชัน** ในเมนูที่ซับซ้อน

3. **ปิดโหมด debug:**
   ```yaml
   craftmenu:
     general:
       debug: false
   ```

4. **เพิ่ม update interval:**
   ```yaml
   craftmenu:
     performance:
       update-interval: 2
   ```

## ปลั๊กอินไม่โหลด

**อาการ:** ปลั๊กอินแสดงข้อผิดพลาดตอนเริ่มต้น

**วิธีแก้ไข:**

1. **ตรวจสอบเวอร์ชัน Java:**
   - ต้องการ Java 17 หรือสูงกว่า

2. **ตรวจสอบ dependencies:**
   - PacketEvents ต้องติดตั้งแล้ว

3. **ตรวจสอบเวอร์ชันเซิร์ฟเวอร์:**
   - ต้องการ Minecraft 1.20.4+

4. **ตรวจสอบ log เมื่อเริ่มต้น** สำหรับข้อผิดพลาดเฉพาะ

5. **ลองกู้คืน:**
   ```
   /cm fukfuen
   ```

## ข้อผิดพลาด YAML

**อาการ:** ข้อผิดพลาดกล่าวถึงการ parse YAML

**ปัญหาทั่วไป:**

1. **การย่อหน้าไม่ถูกต้อง:**
   ```yaml
   # ผิด
   widgets:
   my_widget:
     type: IMAGE

   # ถูก
   widgets:
     my_widget:
       type: IMAGE
   ```

2. **ขาดเครื่องหมายคำพูดรอบค่าพิเศษ:**
   ```yaml
   # ผิด - & มีความหมายพิเศษ
   title: &bHello

   # ถูก
   title: "&bHello"
   ```

3. **รูปแบบ list ไม่ถูกต้อง:**
   ```yaml
   # ผิด
   events:
     on_cursor_click:
       action: sound

   # ถูก
   events:
     on_cursor_click:
       - action: sound
   ```

## ขอความช่วยเหลือ

ถ้าคุณยังมีปัญหา:

1. เปิดโหมด debug และตรวจสอบ output ของคอนโซล
2. ตรวจสอบ GitHub issues สำหรับปัญหาที่รู้จัก
3. สร้าง issue ใหม่พร้อมด้วย:
   - เวอร์ชันเซิร์ฟเวอร์
   - เวอร์ชันปลั๊กอิน
   - log ของคอนโซล
   - ไฟล์คอนฟิก (ลบข้อมูลสำคัญออก)
