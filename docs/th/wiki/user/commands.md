# คำสั่งทั้งหมด

CraftMenu มีชุดคำสั่งที่ครบครันสำหรับการจัดการเมนู

## คำสั่งหลัก

คำสั่งทั้งหมดใช้ `/craftmenu` (ชื่อย่อ: `/cm`)

## คำสั่งทั่วไป

### ช่วยเหลือ
```
/cm chuayluea [คำสั่ง]
```
แสดงข้อมูลช่วยเหลือสำหรับคำสั่งทั้งหมดหรือคำสั่งเฉพาะ

### รายการเมนู
```
/cm raykan
```
แสดงรายการเทมเพลตเมนูที่โหลดทั้งหมด

### ข้อมูลปลั๊กอิน
```
/cm khomul
```
แสดงเวอร์ชันปลั๊กอินและสถิติ

## คำสั่งเมนู

### เปิดเมนู
```
/cm perd <ชื่อเมนู> [ผู้เล่น]
```
เปิดเมนูสำหรับตัวเองหรือผู้เล่นอื่น

**ตัวอย่าง:**
- `/cm perd template` - เปิดเมนู template สำหรับตัวเอง
- `/cm perd lobby Steve` - เปิดเมนู lobby สำหรับผู้เล่น Steve

### ปิดเมนู
```
/cm pid [ผู้เล่น]
```
ปิดเมนูที่เปิดอยู่สำหรับตัวเองหรือผู้เล่นอื่น

### สร้างเมนู
```
/cm sangmai <ชื่อเมนู>
```
สร้างเทมเพลตเมนูใหม่ที่ตำแหน่งปัจจุบันของคุณ

### ลบเมนู
```
/cm lob <ชื่อเมนู>
```
ลบเทมเพลตเมนู

## คำสั่ง Resource Pack

### สร้าง Resource Pack
```
/cm pakked
```
สร้าง resource pack จากภาพและเสียงในโฟลเดอร์ CraftMenu

### คำสั่งภาพ
```
/cm phap truat
/cm phap kaekhai [--backup]
/cm phap resize <เส้นทางภาพ> <ขนาดเป้าหมาย>
/cm phap samrong
/cm phap kuen <ชื่อ backup>
/cm phap raykan
/cm phap samrongtang
```
- `truat` - สแกนหาภาพที่มีขนาดใหญ่เกินไป
- `kaekhai` - ปรับแต่งภาพที่มีขนาดใหญ่เกินไปโดยอัตโนมัติ
- `resize` - ปรับขนาดภาพเฉพาะเป็นขนาดเป้าหมาย (16-4096 พิกเซล)
- `samrong` - สร้าง backup ของภาพ
- `kuen` - คืนค่าภาพจาก backup
- `raykan` - แสดงรายการภาพทั้งหมดในโฟลเดอร์ images
- `samrongtang` - แสดงรายการ backup ที่มีทั้งหมด

## คำสั่งคอนฟิก

### โหลดใหม่
```
/cm lodmai
```
โหลดคอนฟิกและเทมเพลตเมนูทั้งหมดใหม่

### ภาษา
```
/cm phasa <ภาษา>
/cm phasa raykan
```
เปลี่ยนภาษาปลั๊กอิน ไม่ต้องใช้ "tang" อีกต่อไป - ใช้แค่ `/cm phasa <ภาษา>`

- `/cm phasa <ภาษา>` - ตั้งค่าภาษาที่ระบุ
- `/cm phasa raykan` - แสดงรายการภาษาที่รองรับทั้งหมด

ภาษาที่รองรับ:
- `en_US` - อังกฤษ
- `pt_BR` - โปรตุเกส (บราซิล)
- `es_ES` - สเปน
- `fr_FR` - ฝรั่งเศส
- `de_DE` - เยอรมัน
- `it_IT` - อิตาลี
- `nl_NL` - ดัตช์
- `ru_RU` - รัสเซีย
- `pl_PL` - โปแลนด์
- `tr_TR` - ตุรกี
- `uk_UA` - ยูเครน
- `ar_SA` - อาหรับ
- `ja_JP` - ญี่ปุ่น
- `ko_KR` - เกาหลี
- `zh_CN` - จีน (ตัวย่อ)
- `hi_IN` - ฮินดี
- `id_ID` - อินโดนีเซีย
- `th_TH` - ไทย
- `vi_VN` - เวียดนาม

## คำสั่ง Debug

### Debug อนุภาค
```
/cm kaekhong anupak
/cm kaekhong anupak khanad <ค่า>
```
- `/cm kaekhong anupak` - สลับอนุภาค debug ทั้งหมด (การชน, เคอร์เซอร์, จุดศูนย์กลาง)
- `/cm kaekhong anupak khanad <ค่า>` - ตั้งค่าขนาดอนุภาค (0.001 ถึง 2.0)

### Debug ตาราง
```
/cm kaekhong tarang
/cm kaekhong tarang lek
```
- `/cm kaekhong tarang` - สลับการแสดงตาราง debug
- `/cm kaekhong tarang lek` - สลับการแสดงตัวเลขช่อง

### ตรวจสอบสุขภาพ
```
/cm sukhaphap
```
แสดงสถานะสุขภาพของส่วนประกอบ

### กู้คืน
```
/cm fukfuen
```
พยายามกู้คืนจากข้อผิดพลาด

## คำสั่งตัวแก้ไข

เปิดตัวแก้ไขภาพในเกมสำหรับเมนูและวิดเจ็ต

### เปิดตัวแก้ไข
```
/cm kaikai
/cm kaikai <ชื่อเมนู>
```
- `/cm kaikai` - เปิดหน้าหลักตัวแก้ไข
- `/cm kaikai <ชื่อเมนู>` - เปิดตัวแก้ไขสำหรับเมนูที่ระบุ

**สิทธิ์ที่ต้องการ:** `craftmenu.admin`

## สิทธิ์

| สิทธิ์ | คำอธิบาย |
|--------|----------|
| `craftmenu.use` | การใช้งานพื้นฐาน (เปิดเมนู) |
| `craftmenu.admin` | คำสั่งผู้ดูแลระบบ |
| `craftmenu.open` | เปิดเมนู |
| `craftmenu.create` | สร้างเมนู |
| `craftmenu.reload` | โหลดปลั๊กอินใหม่ |
| `craftmenu.debug` | คำสั่ง debug |
