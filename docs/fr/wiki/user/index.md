# Guide Utilisateur CraftMenu

Bienvenue dans le Guide Utilisateur CraftMenu ! CraftMenu est un systeme de menu 3D avance pour Minecraft qui vous permet de creer des menus immersifs et interactifs rendus dans le monde du jeu.

## Qu'est-ce que CraftMenu ?

CraftMenu transforme la facon dont les joueurs interagissent avec les menus dans Minecraft. Au lieu des menus traditionnels bases sur l'inventaire, CraftMenu rend de beaux menus 3D en utilisant des entites TextDisplay qui flottent dans le monde du jeu.

## Fonctionnalites Principales

- **Rendu de Menu 3D** - Les menus sont rendus dans le monde du jeu en utilisant des entites TextDisplay
- **Interaction par Souris** - Controle complet du curseur avec detection du survol et des clics
- **Generation Automatique de Resource Pack** - Generation automatique de resource packs pour les images et sons personnalises
- **Systeme d'Animation** - 19 types d'animations avec 6 fonctions d'acceleration
- **Support Multi-langues** - Support intégré pour 20 langues (anglais, portugais brésilien, portugais européen, espagnol, français, allemand, italien, néerlandais, russe, polonais, turc, ukrainien, arabe, japonais, coréen, chinois, hindi, indonésien, thaï, vietnamien)
- **Integration PlaceholderAPI** - Affichez des donnees dynamiques des joueurs dans vos menus
- **Ouverture Automatique des Menus** - Ouvrez automatiquement les menus quand les joueurs rejoignent ou se teleportent

## Demarrage Rapide

1. **Installez le plugin** - Placez le JAR dans votre dossier `plugins/`
2. **Demarrez votre serveur** - CraftMenu generera les configurations par defaut
3. **Generez le resource pack** - Executez `/cm zip` pour generer le resource pack
4. **Ouvrez un menu** - Utilisez `/cm open template` pour tester le menu par defaut
