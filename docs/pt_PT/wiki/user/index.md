# Guia do Utilizador CraftMenu

Bem-vindo ao Guia do Utilizador do CraftMenu! CraftMenu é um sistema avançado de menus 3D para Minecraft que permite criar menus imersivos e interativos renderizados no mundo do jogo.

## O que é CraftMenu?

CraftMenu transforma a forma como os jogadores interagem com menus no Minecraft. Ao invés de menus tradicionais baseados em inventários, CraftMenu renderiza belos menus 3D usando entidades TextDisplay que flutuam no mundo do jogo.

## Recursos Principais

- **Renderização de Menu 3D** - Menus são renderizados no mundo do jogo usando entidades TextDisplay
- **Interação por Rato** - Controlo total do cursor com deteção de hover e clique
- **Geração Automática de Resource Pack** - Geração automática de resource packs para imagens e sons personalizados
- **Sistema de Animações** - 19 tipos de animação com 6 funções de easing
- **Suporte Multi-idioma** - Suporte integrado para 20 idiomas (Inglês, Português Brasil, Português Portugal, Espanhol, Francês, Alemão, Italiano, Holandês, Russo, Polaco, Turco, Ucraniano, Árabe, Japonês, Coreano, Chinês, Hindi, Indonésio, Tailandês, Vietnamita)
- **Integração PlaceholderAPI** - Exiba dados dinâmicos do jogador nos seus menus
- **Menus de Abertura Automática** - Abra menus automaticamente quando jogadores entrarem ou teleportarem

## Início Rápido

1. **Instale o plugin** - Coloque o JAR na pasta `plugins/`
2. **Inicie o seu servidor** - CraftMenu gerará as configurações padrão
3. **Gere o resource pack** - Execute `/cm pacote` para gerar o resource pack
4. **Abra um menu** - Use `/cm abrir template` para testar o menu padrão
