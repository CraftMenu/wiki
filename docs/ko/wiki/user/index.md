# CraftMenu 사용자 가이드

CraftMenu 사용자 가이드에 오신 것을 환영합니다! CraftMenu는 게임 월드에서 렌더링되는 몰입형 인터랙티브 메뉴를 생성할 수 있는 고급 3D 메뉴 시스템입니다.

## CraftMenu란 무엇인가요?

CraftMenu는 마인크래프트에서 플레이어가 메뉴와 상호작용하는 방식을 혁신합니다. 전통적인 인벤토리 기반 메뉴 대신, CraftMenu는 게임 월드에 떠 있는 TextDisplay 엔티티를 사용하여 아름다운 3D 메뉴를 렌더링합니다.

## 주요 기능

- **3D 메뉴 렌더링** - TextDisplay 엔티티를 사용하여 게임 월드에 메뉴 렌더링
- **마우스 기반 상호작용** - 호버 및 클릭 감지 기능이 있는 완전한 커서 제어
- **사용자 정의 리소스팩 생성** - 사용자 정의 이미지 및 사운드를 위한 리소스팩 자동 생성
- **애니메이션 시스템** - 6가지 이징 함수가 있는 19가지 애니메이션 유형
- **다국어 지원** - 20개 언어 내장 지원 (영어, 브라질 포르투갈어, 유럽 포르투갈어, 스페인어, 프랑스어, 독일어, 이탈리아어, 네덜란드어, 러시아어, 폴란드어, 터키어, 우크라이나어, 아랍어, 일본어, 한국어, 중국어, 힌디어, 인도네시아어, 태국어, 베트남어)
- **PlaceholderAPI 통합** - 메뉴에 동적 플레이어 데이터 표시
- **자동 열기 메뉴** - 플레이어가 접속하거나 텔레포트할 때 자동으로 메뉴 열기

## 빠른 시작

1. **플러그인 설치** - JAR 파일을 `plugins/` 폴더에 넣기
2. **서버 시작** - CraftMenu가 기본 구성 파일 생성
3. **리소스팩 생성** - `/cm paekij` 실행하여 리소스팩 생성
4. **메뉴 열기** - `/cm yeolda template`을 사용하여 기본 메뉴 테스트
