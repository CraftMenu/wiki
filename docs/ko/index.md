# CraftMenu Wiki에 오신 것을 환영합니다

**CraftMenu**는 Minecraft용 전문 3D 메뉴 시스템으로, 월드의 3D 공간에서 완전히 상호작용하는 메뉴를 만들 수 있습니다.

- **[기능](FEATURES.md)** - CraftMenu의 모든 기능 탐색
- **[빠른 시작](QUICKSTART.md)** - 몇 분 안에 시작
- **[사용자 가이드](wiki/user/installation.md)** - 전체 문서
- **[메뉴 생성](MENU_CREATION.md)** - 메뉴 만드는 법 배우기

## 📚 여기서 찾을 수 있는 것

- **설치 가이드** - 단계별 지침
- **구성** - 메뉴를 구성하는 방법 배우기
- **위젯** - 위젯 시스템 이해하기
- **애니메이션** - 놀라운 시각 효과 만들기
- **이벤트** - 상호작용 동작 구현
- **문제 해결** - 일반적인 문제 해결

## 🎯 주요 기능

- ✨ **3D 메뉴 렌더링** - 월드 공간의 아름다운 메뉴
- 🖱️ **마우스 상호작용** - 완전한 커서 제어
- 🎬 **19가지 애니메이션 유형** - 역동적인 경험 만들기
- 🌐 **20개 언어** - 다국어 지원
- 🎨 **사용자 정의 텍스처** - 자신만의 이미지 사용
- 🔊 **사운드 시스템** - 오디오 피드백 추가
