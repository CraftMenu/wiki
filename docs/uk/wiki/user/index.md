# Посібник користувача CraftMenu

Ласкаво просимо до посібника користувача CraftMenu! CraftMenu - це просунута система 3D меню для Minecraft, яка дозволяє створювати занурювальні, інтерактивні меню, що рендеряться у світі гри.

## Що таке CraftMenu?

CraftMenu трансформує спосіб взаємодії гравців з меню в Minecraft. Замість традиційних меню на основі інвентарю, CraftMenu рендерить красиві 3D меню, використовуючи TextDisplay сутності, які парять у світі гри.

## Основні можливості

- **Рендеринг 3D меню** - Меню рендеряться у світі гри за допомогою TextDisplay сутностей
- **Взаємодія мишею** - Повний контроль курсора з визначенням наведення та кліків
- **Автоматична генерація ресурс-паку** - Автоматична генерація ресурс-паків для кастомних зображень та звуків
- **Система анімацій** - 19 типів анімацій з 6 функціями згладжування
- **Підтримка багатьох мов** - Вбудована підтримка 20 мов (англійська, португальська (Бразилія), португальська (Португалія), іспанська, французька, німецька, італійська, нідерландська, російська, польська, турецька, українська, арабська, японська, корейська, китайська, гінді, індонезійська, тайська, в'єтнамська)
- **Інтеграція PlaceholderAPI** - Відображення динамічних даних гравця у ваших меню
- **Автовідкриття меню** - Автоматичне відкриття меню при вході гравців або телепортації

## Швидкий старт

1. **Встановіть плагін** - Помістіть JAR у папку `plugins/`
2. **Запустіть сервер** - CraftMenu згенерує стандартні конфігурації
3. **Згенеруйте ресурс-пак** - Виконайте `/cm zip` для генерації ресурс-паку
4. **Відкрийте меню** - Використайте `/cm open template` для тестування стандартного меню
