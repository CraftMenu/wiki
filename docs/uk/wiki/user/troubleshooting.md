# Вирішення проблем

Поширені проблеми та рішення для CraftMenu.

## Зображення не відображаються

**Симптом:** Зображення показуються як "?" або відсутні символи.

**Рішення:**

1. **Перегенеруйте ресурс-пак:**
   ```
   /cm zip
   ```

2. **Перевірте розташування зображення:**
   - Зображення повинні бути в підпапках: `plugins/CraftMenu/images/folder/image.png`
   - НЕ в корені: `plugins/CraftMenu/images/image.png`

3. **Перевірте формат зображення:**
   - Підтримуються тільки PNG файли
   - Переконайтеся в правильному розширенні файлу (`.png`, не `.PNG`)

4. **Перевірте що ресурс-пак завантажений:**
   - Серверний ресурс-пак повинен бути налаштований
   - Гравець повинен прийняти ресурс-пак

5. **Перезавантажте плагін:**
   ```
   /cm reload
   ```

## Меню не відкривається

**Симптом:** Команда `/cm open` нічого не робить.

**Рішення:**

1. **Перевірте що меню існує:**
   ```
   /cm list
   ```

2. **Перевірте консоль на помилки** після виконання команди

3. **Перевірте синтаксис YAML:**
   - Використовуйте YAML валідатор
   - Перевірте правильність відступів

4. **Переконайтеся що локація спавну валідна:**
   - Світ повинен бути завантажений
   - Локація повинна бути доступною

## Колізія не працює

**Симптом:** Курсор не виявляє віджети.

**Рішення:**

1. **Увімкніть частинки налагодження:**
   ```
   /debugcollision toggle
   ```

2. **Перевірте конфігурацію колізії:**
   ```yaml
   collision:
     enabled: true
     size: {x: 0.1, y: 0.1, z: 0.1}
   ```

3. **Збільшіть розмір коробки колізії** якщо вона занадто мала

4. **Перевірте позицію віджета** - колізія може бути зміщена

## Звуки не відтворюються

**Симптом:** Дії звуку не мають ефекту.

**Рішення:**

1. **Для кастомних звуків:**
   - Розмістіть `.ogg` файли в `plugins/CraftMenu/sounds/folder/`
   - Перегенеруйте ресурс-пак: `/cm zip`

2. **Для звуків Minecraft:**
   - Використовуйте правильний формат: `minecraft:ui.button.click`

3. **Перевірте налаштування гучності** в конфігурації дії

## Проблеми продуктивності

**Симптом:** Лаг при використанні меню.

**Рішення:**

1. **Оптимізуйте зображення:**
   ```
   /cm images scan
   /cm images fix --backup
   ```

2. **Зменшіть частоту анімацій** у складних меню

3. **Вимкніть режим налагодження:**
   ```yaml
   craftmenu:
     general:
       debug: false
   ```

4. **Збільшіть інтервал оновлення:**
   ```yaml
   craftmenu:
     performance:
       update-interval: 2
   ```

## Плагін не завантажується

**Симптом:** Плагін показує помилки при запуску.

**Рішення:**

1. **Перевірте версію Java:**
   - Потрібна Java 17 або вище

2. **Перевірте залежності:**
   - PacketEvents повинен бути встановлений

3. **Перевірте версію сервера:**
   - Потрібен Minecraft 1.20.4+

4. **Перегляньте логи запуску** на конкретні помилки

5. **Спробуйте відновлення:**
   ```
   /cm recover
   ```

## Помилки YAML

**Симптом:** Помилки згадують парсинг YAML.

**Поширені проблеми:**

1. **Неправильні відступи:**
   ```yaml
   # Неправильно
   widgets:
   my_widget:
     type: IMAGE

   # Правильно
   widgets:
     my_widget:
       type: IMAGE
   ```

2. **Відсутні лапки навколо спеціальних значень:**
   ```yaml
   # Неправильно - & має спеціальне значення
   title: &bПривіт

   # Правильно
   title: "&bПривіт"
   ```

3. **Неправильний формат списку:**
   ```yaml
   # Неправильно
   events:
     on_cursor_click:
       action: sound

   # Правильно
   events:
     on_cursor_click:
       - action: sound
   ```

## Отримання допомоги

Якщо у вас все ще є проблеми:

1. Увімкніть режим налагодження та перевірте вивід консолі
2. Перевірте GitHub issues на відомі проблеми
3. Створіть новий issue з:
   - Версією сервера
   - Версією плагіна
   - Логами консолі
   - Конфігураційними файлами (видаліть чутливі дані)
