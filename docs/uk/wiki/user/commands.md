# Довідник команд

CraftMenu надає комплексний набір команд для керування меню.

## Базова команда

Всі команди використовують `/craftmenu` (аліас: `/cm`).

## Загальні команди

### Допомога
```
/cm dopomoha [command]
```
Показує інформацію про всі команди або конкретну команду.

### Список меню
```
/cm spysok
```
Показує список всіх завантажених шаблонів меню.

### Інформація про плагін
```
/cm info
```
Показує версію плагіна та статистику.

## Команди меню

### Відкрити меню
```
/cm vidkryty <menu_name> [player]
```
Відкриває меню для вас або іншого гравця.

**Приклади:**
- `/cm vidkryty template` - Відкрити шаблонне меню для себе
- `/cm vidkryty lobby Steve` - Відкрити лобі меню для гравця Steve

### Закрити меню
```
/cm zakryty [player]
```
Закриває активне меню для вас або іншого гравця.

### Створити меню
```
/cm stvoryty <menu_name>
```
Створює новий шаблон меню у вашій поточній локації.

### Видалити меню
```
/cm vydalyty <menu_name>
```
Видаляє шаблон меню.

## Команди ресурс-паку

### Генерація ресурс-паку
```
/cm paket
```
Генерує ресурс-пак із зображень та звуків у папці CraftMenu.

### Команди зображень
```
/cm zobrazhennya skanuvaty
/cm zobrazhennya vypravyty [--backup]
/cm zobrazhennya rozmir <image_path> <target_size>
/cm zobrazhennya rezerv
/cm zobrazhennya vidnovyty <backup_name>
/cm zobrazhennya spysok
/cm zobrazhennya rezervy
```
- `skanuvaty` - Сканує на завеликі зображення
- `vypravyty` - Автоматично оптимізує завеликі зображення
- `rozmir` - Змінює розмір конкретного зображення до цільового розміру (16-4096 пікселів)
- `rezerv` - Створює резервну копію зображень
- `vidnovyty` - Відновлює зображення з резервної копії
- `spysok` - Показує список всіх зображень у папці images
- `rezervy` - Показує список всіх доступних резервних копій

## Команди конфігурації

### Перезавантаження
```
/cm perezavantazhyty
```
Перезавантажує всі конфігурації та шаблони меню.

### Мова
```
/cm mova <lang>
/cm mova spysok
```
- `/cm mova <lang>` - Змінює мову плагіна напряму (без "set")
- `/cm mova spysok` - Показує список доступних мов

**Доступні мови:**
- `en_US` - Англійська
- `pt_BR` - Португальська (Бразилія)
- `es_ES` - Іспанська
- `fr_FR` - Французька
- `de_DE` - Німецька
- `it_IT` - Італійська
- `nl_NL` - Нідерландська
- `ru_RU` - Російська
- `pl_PL` - Польська
- `tr_TR` - Турецька
- `uk_UA` - Українська
- `ar_SA` - Арабська
- `ja_JP` - Японська
- `ko_KR` - Корейська
- `zh_CN` - Китайська (спрощена)
- `hi_IN` - Гінді
- `id_ID` - Індонезійська
- `th_TH` - Тайська
- `vi_VN` - В'єтнамська

## Команди налагодження

### Частинки налагодження
```
/cm naladka chastynky
/cm naladka chastynky rozmir <значення>
```
- `/cm naladka chastynky` - Перемикає ВСІ частинки налагодження (коробки колізій, позиція курсора, центри віджетів)
- `/cm naladka chastynky rozmir <значення>` - Встановлює розмір частинок (0.001 до 2.0)

### Налагодження сітки
```
/cm naladka sitka
/cm naladka sitka nomery
```
- `/cm naladka sitka` - Перемикає візуалізацію сітки налагодження
- `/cm naladka sitka nomery` - Перемикає відображення номерів комірок сітки

### Перевірка стану
```
/cm zdorovya
```
Показує стан здоров'я компонентів.

### Відновлення
```
/cm vidnovyty
```
Спробує відновитися після помилок.

## Команда редактора

Відкриває візуальний внутрішньоігровий редактор для меню та віджетів.

### Відкрити редактор
```
/cm redaktor
/cm redaktor <назва_меню>
```
- `/cm redaktor` - Відкриває головне меню редактора
- `/cm redaktor <назва_меню>` - Відкриває редактор для конкретного меню

**Необхідний дозвіл:** `craftmenu.admin`

## Дозволи

| Дозвіл | Опис |
|--------|------|
| `craftmenu.use` | Базове використання (відкриття меню) |
| `craftmenu.admin` | Адмін команди |
| `craftmenu.open` | Відкриття меню |
| `craftmenu.create` | Створення меню |
| `craftmenu.reload` | Перезавантаження плагіна |
| `craftmenu.debug` | Команди налагодження |
