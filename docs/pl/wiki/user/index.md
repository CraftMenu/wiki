# Przewodnik uzytkownika CraftMenu

Witamy w Przewodniku uzytkownika CraftMenu! CraftMenu to zaawansowany system menu 3D dla Minecraft, ktory pozwala tworzyc immersyjne, interaktywne menu renderowane w swiecie gry.

## Czym jest CraftMenu?

CraftMenu zmienia sposob, w jaki gracze wchodza w interakcje z menu w Minecraft. Zamiast tradycyjnych menu opartych na ekwipunku, CraftMenu renderuje piekne menu 3D uzywajac encji TextDisplay, ktore unosza sie w swiecie gry.

## Glowne funkcje

- **Renderowanie menu 3D** - Menu sa renderowane w swiecie gry za pomoca encji TextDisplay
- **Interakcja za pomoca myszy** - Pelna kontrola kursora z wykrywaniem najechania i klikniecia
- **Automatyczne generowanie Resource Pack** - Automatyczne generowanie paczek zasobow dla niestandardowych obrazow i dzwiekow
- **System animacji** - 19 typow animacji z 6 funkcjami wygładzania
- **Wsparcie wielojęzyczne** - Wbudowane wsparcie dla 20 języków (angielski, portugalski brazylijski, portugalski europejski, hiszpański, francuski, niemiecki, włoski, holenderski, rosyjski, polski, turecki, ukraiński, arabski, japoński, koreański, chiński, hindi, indonezyjski, tajski, wietnamski)
- **Integracja z PlaceholderAPI** - Wyswietlanie dynamicznych danych gracza w menu
- **Automatyczne otwieranie menu** - Automatyczne otwieranie menu gdy gracze dolaczaja lub teleportuja sie

## Szybki start

1. **Zainstaluj plugin** - Umiesc plik JAR w folderze `plugins/`
2. **Uruchom serwer** - CraftMenu wygeneruje domyslne konfiguracje
3. **Wygeneruj resource pack** - Uruchom `/cm zip` aby wygenerowac paczke zasobow
4. **Otworz menu** - Uzyj `/cm open template` aby przetestowac domyslne menu
