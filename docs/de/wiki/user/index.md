# CraftMenu Benutzerhandbuch

Willkommen zum CraftMenu Benutzerhandbuch! CraftMenu ist ein fortschrittliches 3D-Menuesystem fuer Minecraft, mit dem Sie immersive, interaktive Menues erstellen koennen, die in der Spielwelt gerendert werden.

## Was ist CraftMenu?

CraftMenu veraendert die Art und Weise, wie Spieler mit Menues in Minecraft interagieren. Anstatt traditionelle inventarbasierte Menues zu verwenden, rendert CraftMenu wunderschoene 3D-Menues mithilfe von TextDisplay-Entities, die in der Spielwelt schweben.

## Hauptfunktionen

- **3D-Menuedarstellung** - Menues werden in der Spielwelt mithilfe von TextDisplay-Entities gerendert
- **Mausbasierte Interaktion** - Vollstaendige Cursorsteuerung mit Hover- und Klick-Erkennung
- **Automatische Ressourcenpaket-Generierung** - Automatische Erstellung von Ressourcenpaketen fuer benutzerdefinierte Bilder und Sounds
- **Animationssystem** - 19 Animationstypen mit 6 Easing-Funktionen
- **Mehrsprachige Unterstützung** - Integrierte Unterstützung für 20 Sprachen (Englisch, Portugiesisch (Brasilien), Portugiesisch (Portugal), Spanisch, Französisch, Deutsch, Italienisch, Niederländisch, Russisch, Polnisch, Türkisch, Ukrainisch, Arabisch, Japanisch, Koreanisch, Chinesisch, Hindi, Indonesisch, Thai, Vietnamesisch)
- **PlaceholderAPI-Integration** - Zeigen Sie dynamische Spielerdaten in Ihren Menues an
- **Auto-Open Menues** - Oeffnen Sie Menues automatisch, wenn Spieler beitreten oder teleportiert werden

## Schnellstart

1. **Plugin installieren** - Legen Sie die JAR-Datei in Ihren `plugins/`-Ordner
2. **Server starten** - CraftMenu generiert die Standardkonfigurationen
3. **Ressourcenpaket generieren** - Fuehren Sie `/cm paket` aus, um das Ressourcenpaket zu generieren
4. **Menue oeffnen** - Verwenden Sie `/cm oeffnen template`, um das Standardmenue zu testen
