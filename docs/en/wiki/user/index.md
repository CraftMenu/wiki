# CraftMenu User Guide

Welcome to the CraftMenu User Guide! CraftMenu is an advanced 3D menu system for Minecraft that allows you to create immersive, interactive menus rendered in the game world.

## What is CraftMenu?

CraftMenu transforms how players interact with menus in Minecraft. Instead of traditional inventory-based menus, CraftMenu renders beautiful 3D menus using TextDisplay entities that float in the game world.

## Key Features

- **3D Menu Rendering** - Menus are rendered in the game world using TextDisplay entities
- **Mouse-Based Interaction** - Full cursor control with hover and click detection
- **Custom Resource Pack Generation** - Automatic generation of resource packs for custom images and sounds
- **Animation System** - 19 animation types with 6 easing functions
- **Multi-Language Support** - Built-in support for 20 languages (English, Portuguese Brazil, Portuguese Portugal, Spanish, French, German, Italian, Dutch, Russian, Polish, Turkish, Ukrainian, Arabic, Japanese, Korean, Chinese, Hindi, Indonesian, Thai, Vietnamese)
- **PlaceholderAPI Integration** - Display dynamic player data in your menus
- **Auto-Open Menus** - Automatically open menus when players join or teleport

## Quick Start

1. **Install the plugin** - Place the JAR in your `plugins/` folder
2. **Start your server** - CraftMenu will generate default configurations
3. **Generate resource pack** - Run `/cm zip` to generate the resource pack
4. **Open a menu** - Use `/cm open template` to test the default menu
