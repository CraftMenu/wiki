# CraftMenu Kullanici Rehberi

CraftMenu Kullanici Rehberine hos geldiniz! CraftMenu, Minecraft'ta oyun dunyasinda render edilen sürükleyici, interaktif menuler olusturmaniza olanak taniyan gelismis bir 3D menu sistemidir.

## CraftMenu Nedir?

CraftMenu, oyuncularin Minecraft'taki menülerle etkilesim bicimini donusturur. Geleneksel envanter tabanli menuler yerine, CraftMenu oyun dunyasinda suzulen TextDisplay entity'lerini kullanarak guzel 3D menuler render eder.

## Temel Ozellikler

- **3D Menu Render** - Menuler oyun dunyasinda TextDisplay entity'leri kullanilarak render edilir
- **Fare Tabanli Etkilesim** - Hover ve tiklama algilamali tam imlec kontrolu
- **Ozel Kaynak Paketi Olusturma** - Ozel gorseller ve sesler icin otomatik kaynak paketi olusturma
- **Animasyon Sistemi** - 6 easing fonksiyonuyla 19 animasyon turu
- **Çoklu Dil Desteği** - 20 dil için yerleşik destek (İngilizce, Portekizce (Brezilya), Portekizce (Portekiz), İspanyolca, Fransızca, Almanca, İtalyanca, Hollandaca, Rusça, Lehçe, Türkçe, Ukraynaca, Arapça, Japonca, Korece, Çince, Hintçe, Endonezce, Tayca, Vietnamca)
- **PlaceholderAPI Entegrasyonu** - Menulerinizde dinamik oyuncu verileri gosterin
- **Otomatik Menu Acilisi** - Oyuncular katildiginda veya isinlandiginda menuleri otomatik acin

## Hizli Baslangic

1. **Eklentiyi yukleyin** - JAR dosyasini `plugins/` klasorunuze koyun
2. **Sunucunuzu baslatin** - CraftMenu varsayilan yapilandirmalari olusturacak
3. **Kaynak paketi olusturun** - Kaynak paketini olusturmak icin `/cm zip` calistirin
4. **Bir menu acin** - Varsayilan menuyu test etmek icin `/cm open template` kullanin
