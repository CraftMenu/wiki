# CraftMenu ユーザーガイド

CraftMenuユーザーガイドへようこそ！CraftMenuは、ゲーム世界内でレンダリングされる没入型のインタラクティブメニューを作成できる、Minecraft用の高度な3Dメニューシステムです。

## CraftMenuとは？

CraftMenuは、Minecraftにおけるプレイヤーのメニュー操作方法を変革します。従来のインベントリベースのメニューとは異なり、CraftMenuはゲーム世界に浮かぶTextDisplayエンティティを使用して美しい3Dメニューをレンダリングします。

## 主な機能

- **3Dメニューレンダリング** - TextDisplayエンティティを使用してゲーム世界内にメニューがレンダリングされます
- **マウスベースのインタラクション** - ホバーとクリック検出を備えた完全なカーソル制御
- **カスタムリソースパック生成** - カスタム画像とサウンド用のリソースパックの自動生成
- **アニメーションシステム** - 6つのイージング関数を持つ19種類のアニメーションタイプ
- **多言語サポート** - 20言語の組み込みサポート（英語、ブラジルポルトガル語、ヨーロッパポルトガル語、スペイン語、フランス語、ドイツ語、イタリア語、オランダ語、ロシア語、ポーランド語、トルコ語、ウクライナ語、アラビア語、日本語、韓国語、中国語、ヒンディー語、インドネシア語、タイ語、ベトナム語）
- **PlaceholderAPI連携** - メニューに動的なプレイヤーデータを表示
- **自動オープンメニュー** - プレイヤーの参加時やテレポート時にメニューを自動的に開く

## クイックスタート

1. **プラグインをインストール** - JARファイルを`plugins/`フォルダに配置
2. **サーバーを起動** - CraftMenuがデフォルト設定を生成します
3. **リソースパックを生成** - `/cm pakku`を実行してリソースパックを生成
4. **メニューを開く** - `/cm hiraku template`を使用してデフォルトメニューをテスト
