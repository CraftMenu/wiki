# CraftMenu Wikiへようこそ

**CraftMenu**は、Minecraftのプロフェッショナルな3Dメニューシステムで、ワールドの3D空間に完全にインタラクティブなメニューを作成できます。

- **[クイックスタート](QUICKSTART.md)** - 数分で開始
- **[ユーザーガイド](wiki/user/installation.md)** - 完全なドキュメント
- **[メニュー作成](MENU_CREATION.md)** - メニューの作成方法を学ぶ

## 📚 ここで見つけられるもの

- **インストールガイド** - ステップバイステップの手順
- **設定** - メニューの設定方法を学ぶ
- **ウィジェット** - ウィジェットシステムを理解する
- **アニメーション** - 素晴らしいビジュアル効果を作成
- **イベント** - インタラクティブな動作を実装
- **トラブルシューティング** - 一般的な問題を解決

## 🎯 主な機能

- ✨ **3Dメニューレンダリング** - ワールド空間の美しいメニュー
- 🖱️ **マウスインタラクション** - カーソルの完全な制御
- 🎬 **19種類のアニメーション** - ダイナミックな体験を作成
- 🌐 **20言語** - 多言語サポート
- 🎨 **カスタムテクスチャ** - 独自の画像を使用
- 🔊 **サウンドシステム** - オーディオフィードバックを追加
