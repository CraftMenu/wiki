# Guida Utente CraftMenu

Benvenuto nella Guida Utente di CraftMenu! CraftMenu e un sistema avanzato di menu 3D per Minecraft che ti permette di creare menu immersivi e interattivi renderizzati nel mondo di gioco.

## Cos'e CraftMenu?

CraftMenu trasforma il modo in cui i giocatori interagiscono con i menu in Minecraft. Invece dei tradizionali menu basati sull'inventario, CraftMenu renderizza bellissimi menu 3D utilizzando entita TextDisplay che fluttuano nel mondo di gioco.

## Caratteristiche Principali

- **Rendering Menu 3D** - I menu sono renderizzati nel mondo di gioco usando entita TextDisplay
- **Interazione con Mouse** - Controllo completo del cursore con rilevamento hover e click
- **Generazione Automatica Resource Pack** - Generazione automatica di resource pack per immagini e suoni personalizzati
- **Sistema di Animazioni** - 19 tipi di animazione con 6 funzioni di easing
- **Supporto Multi-Lingua** - Supporto integrato per 20 lingue (Inglese, Portoghese Brasiliano, Portoghese Europeo, Spagnolo, Francese, Tedesco, Italiano, Olandese, Russo, Polacco, Turco, Ucraino, Arabo, Giapponese, Coreano, Cinese, Hindi, Indonesiano, Tailandese, Vietnamita)
- **Integrazione PlaceholderAPI** - Visualizza dati dinamici dei giocatori nei tuoi menu
- **Menu Auto-Open** - Apri automaticamente i menu quando i giocatori entrano o si teletrasportano

## Avvio Rapido

1. **Installa il plugin** - Posiziona il JAR nella cartella `plugins/`
2. **Avvia il server** - CraftMenu generera le configurazioni predefinite
3. **Genera il resource pack** - Esegui `/cm pacchetto` per generare il resource pack
4. **Apri un menu** - Usa `/cm apri template` per testare il menu predefinito
