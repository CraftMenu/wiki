# Руководство пользователя CraftMenu

Добро пожаловать в руководство пользователя CraftMenu! CraftMenu - это продвинутая система 3D-меню для Minecraft, которая позволяет создавать иммерсивные интерактивные меню, отображаемые в игровом мире.

## Что такое CraftMenu?

CraftMenu трансформирует способ взаимодействия игроков с меню в Minecraft. Вместо традиционных меню на основе инвентаря, CraftMenu отображает красивые 3D-меню с использованием сущностей TextDisplay, которые парят в игровом мире.

## Ключевые возможности

- **3D-рендеринг меню** - Меню отображаются в игровом мире с использованием сущностей TextDisplay
- **Управление мышью** - Полный контроль курсора с определением наведения и клика
- **Автоматическая генерация Resource Pack** - Автоматическое создание ресурспаков для пользовательских изображений и звуков
- **Система анимаций** - 19 типов анимаций с 6 функциями сглаживания
- **Многоязычная поддержка** - Встроенная поддержка 20 языков (английский, португальский (Бразилия), португальский (Португалия), испанский, французский, немецкий, итальянский, голландский, русский, польский, турецкий, украинский, арабский, японский, корейский, китайский, хинди, индонезийский, тайский, вьетнамский)
- **Интеграция PlaceholderAPI** - Отображение динамических данных игрока в меню
- **Автооткрытие меню** - Автоматическое открытие меню при входе или телепортации игроков

## Быстрый старт

1. **Установите плагин** - Поместите JAR-файл в папку `plugins/`
2. **Запустите сервер** - CraftMenu создаст конфигурации по умолчанию
3. **Сгенерируйте ресурспак** - Выполните `/cm zip` для создания ресурспака
4. **Откройте меню** - Используйте `/cm open template` для тестирования стандартного меню
