# Справочник команд

CraftMenu предоставляет полный набор команд для управления меню.

## Базовая команда

Все команды используют `/craftmenu` (алиас: `/cm`).

## Общие команды

### Помощь
```
/cm pomosh [command]
```
Показывает справочную информацию по всем командам или конкретной команде.

### Список меню
```
/cm spisok
```
Показывает все загруженные шаблоны меню.

### Информация о плагине
```
/cm info
```
Показывает версию плагина и статистику.

## Команды меню

### Открыть меню
```
/cm otkryt <menu_name> [player]
```
Открывает меню для вас или другого игрока.

**Примеры:**
- `/cm otkryt template` - Открыть шаблонное меню для себя
- `/cm otkryt lobby Steve` - Открыть лобби-меню для игрока Steve

### Закрыть меню
```
/cm zakryt [player]
```
Закрывает активное меню для вас или другого игрока.

### Создать меню
```
/cm sozdat <menu_name>
```
Создает новый шаблон меню в вашем текущем местоположении.

### Удалить меню
```
/cm udalit <menu_name>
```
Удаляет шаблон меню.

## Команды Resource Pack

### Сгенерировать Resource Pack
```
/cm paket
```
Генерирует ресурспак из изображений и звуков в папке CraftMenu.

### Команды изображений
```
/cm izobrazheniya skanirovat
/cm izobrazheniya ispravit [--backup]
/cm izobrazheniya razmer <image_path> <target_size>
/cm izobrazheniya rezerv
/cm izobrazheniya vosstanovit <backup_name>
/cm izobrazheniya spisok
/cm izobrazheniya rezervy
```
- `skanirovat` - Сканирует на предмет слишком больших изображений
- `ispravit` - Автоматически оптимизирует большие изображения
- `razmer` - Изменяет размер определенного изображения до целевого размера (16-4096 пикселей)
- `rezerv` - Создает резервную копию изображений
- `vosstanovit` - Восстанавливает изображения из резервной копии
- `spisok` - Показывает все изображения в папке images
- `rezervy` - Показывает все доступные резервные копии

## Команды конфигурации

### Перезагрузка
```
/cm perezagruzit
```
Перезагружает все конфигурации и шаблоны меню.

### Язык
```
/cm yazyk <lang>
/cm yazyk spisok
```
- `/cm yazyk <lang>` - Изменяет язык плагина напрямую (без "set")
- `/cm yazyk spisok` - Показывает список доступных языков

**Доступные языки:**
- `en_US` - Английский
- `pt_BR` - Португальский (Бразилия)
- `es_ES` - Испанский
- `fr_FR` - Французский
- `de_DE` - Немецкий
- `it_IT` - Итальянский
- `nl_NL` - Голландский
- `ru_RU` - Русский
- `pl_PL` - Польский
- `tr_TR` - Турецкий
- `uk_UA` - Украинский
- `ar_SA` - Арабский
- `ja_JP` - Японский
- `ko_KR` - Корейский
- `zh_CN` - Китайский (упрощенный)
- `hi_IN` - Хинди
- `id_ID` - Индонезийский
- `th_TH` - Тайский
- `vi_VN` - Вьетнамский

## Команды отладки

### Частицы отладки
```
/cm otladka chasticy
/cm otladka chasticy razmer <значение>
```
- `/cm otladka chasticy` - Переключает ВСЕ отладочные частицы (коробки столкновений, положение курсора, центры виджетов)
- `/cm otladka chasticy razmer <значение>` - Устанавливает размер частиц (от 0.001 до 2.0)

### Отладка сетки
```
/cm otladka setka
/cm otladka setka nomera
```
- `/cm otladka setka` - Переключает визуализацию отладочной сетки
- `/cm otladka setka nomera` - Переключает отображение номеров ячеек сетки

### Проверка состояния
```
/cm zdorove
```
Показывает состояние компонентов.

### Восстановление
```
/cm vosstanovit
```
Пытается восстановить работу после ошибок.

## Команда редактора

Открывает визуальный внутриигровой редактор для меню и виджетов.

### Открыть редактор
```
/cm redaktor
/cm redaktor <имя_меню>
```
- `/cm redaktor` - Открывает главное меню редактора
- `/cm redaktor <имя_меню>` - Открывает редактор для конкретного меню

**Требуемое право:** `craftmenu.admin`

## Права доступа

| Право | Описание |
|-------|----------|
| `craftmenu.use` | Базовое использование (открытие меню) |
| `craftmenu.admin` | Административные команды |
| `craftmenu.open` | Открытие меню |
| `craftmenu.create` | Создание меню |
| `craftmenu.reload` | Перезагрузка плагина |
| `craftmenu.debug` | Команды отладки |
