# مرجع الأوامر

يوفر CraftMenu مجموعة شاملة من الأوامر لإدارة القوائم.

## الأمر الأساسي

جميع الأوامر تستخدم `/craftmenu` (الاختصار: `/cm`).

## الأوامر العامة

### المساعدة
```
/cm musaada [الأمر]
```
يعرض معلومات المساعدة لجميع الأوامر أو أمر محدد.

### قائمة القوائم
```
/cm qaima
```
يعرض جميع قوالب القوائم المحملة.

### معلومات الإضافة
```
/cm maalumat
```
يعرض إصدار الإضافة والإحصائيات.

## أوامر القوائم

### فتح القائمة
```
/cm iftah <اسم_القائمة> [اللاعب]
```
يفتح قائمة لنفسك أو للاعب آخر.

**أمثلة:**
- `/cm iftah template` - فتح قائمة template لنفسك
- `/cm iftah lobby Steve` - فتح قائمة lobby للاعب Steve

### إغلاق القائمة
```
/cm ighlaq [اللاعب]
```
يغلق القائمة النشطة لنفسك أو للاعب آخر.

### إنشاء القائمة
```
/cm insha <اسم_القائمة>
```
ينشئ قالب قائمة جديد في موقعك الحالي.

### حذف القائمة
```
/cm hazf <اسم_القائمة>
```
يحذف قالب قائمة.

## أوامر حزمة الموارد

### توليد حزمة الموارد
```
/cm hazma
```
يولد حزمة الموارد من الصور والأصوات في مجلد CraftMenu.

### أوامر الصور
```
/cm suwar maseh
/cm suwar islah [--backup]
/cm suwar taghyir <مسار_الصورة> <الحجم_المستهدف>
/cm suwar ihtiyati
/cm suwar istirda <اسم_النسخة>
/cm suwar qaima
/cm suwar ihtiyatiyat
```
- `maseh` - يفحص الصور كبيرة الحجم
- `islah` - يحسن الصور كبيرة الحجم تلقائيًا
- `taghyir` - يغير حجم صورة معينة إلى الحجم المستهدف (16-4096 بكسل)
- `ihtiyati` - ينشئ نسخة احتياطية من الصور
- `istirda` - يستعيد الصور من نسخة احتياطية
- `qaima` - يعرض جميع الصور في مجلد الصور
- `ihtiyatiyat` - يعرض جميع النسخ الاحتياطية المتاحة

## أوامر التكوين

### إعادة التحميل
```
/cm iadat
```
يعيد تحميل جميع التكوينات وقوالب القوائم.

### اللغة
```
/cm lugha <اللغة>
/cm lugha qaima
```
- `/cm lugha <اللغة>` - يغير لغة الإضافة مباشرة (لا يتطلب "set")
- `/cm lugha qaima` - يعرض جميع اللغات المتاحة

اللغات المتاحة:
- `en_US` - الإنجليزية
- `pt_BR` - البرتغالية (البرازيل)
- `es_ES` - الإسبانية
- `fr_FR` - الفرنسية
- `de_DE` - الألمانية
- `it_IT` - الإيطالية
- `nl_NL` - الهولندية
- `ru_RU` - الروسية
- `pl_PL` - البولندية
- `tr_TR` - التركية
- `uk_UA` - الأوكرانية
- `ar_SA` - العربية
- `ja_JP` - اليابانية
- `ko_KR` - الكورية
- `zh_CN` - الصينية (المبسطة)
- `hi_IN` - الهندية
- `id_ID` - الإندونيسية
- `th_TH` - التايلاندية
- `vi_VN` - الفيتنامية

## أوامر التصحيح

### تصحيح الجسيمات
```
/cm tashih juzayat
/cm tashih juzayat hajm <القيمة>
```
- `/cm tashih juzayat` - يبدل جميع جسيمات التصحيح (صناديق التصادم + مسار المؤشر + مراكز العناصر)
- `/cm tashih juzayat hajm <القيمة>` - يحدد حجم الجسيمات (0.001 إلى 2.0)

### تصحيح الشبكة
```
/cm tashih shabaka
/cm tashih shabaka arqam
```
- `/cm tashih shabaka` - تبديل عرض الشبكة
- `/cm tashih shabaka arqam` - تبديل عرض أرقام الخلايا

### فحص الصحة
```
/cm siha
```
يعرض حالة صحة المكونات.

### الاسترداد
```
/cm istirja
```
يحاول الاسترداد من الأخطاء.

## أمر المحرر

يفتح المحرر المرئي داخل اللعبة للقوائم والعناصر.

### فتح المحرر
```
/cm muharirir
/cm muharirir <اسم_القائمة>
```
- `/cm muharirir` - يفتح واجهة المحرر الرئيسية
- `/cm muharirir <اسم_القائمة>` - يفتح المحرر لقائمة محددة

**الإذن المطلوب:** `craftmenu.admin`

## الأذونات

| الإذن | الوصف |
|-------|-------|
| `craftmenu.use` | الاستخدام الأساسي (فتح القوائم) |
| `craftmenu.admin` | أوامر المسؤول |
| `craftmenu.open` | فتح القوائم |
| `craftmenu.create` | إنشاء القوائم |
| `craftmenu.reload` | إعادة تحميل الإضافة |
| `craftmenu.debug` | أوامر التصحيح |
