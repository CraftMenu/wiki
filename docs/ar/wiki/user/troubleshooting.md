# استكشاف الأخطاء وإصلاحها

المشاكل الشائعة والحلول لـ CraftMenu.

## الصور لا تظهر

**الأعراض:** الصور تظهر كـ "?" أو رموز مفقودة.

**الحلول:**

1. **أعد توليد حزمة الموارد:**
   ```
   /cm hazma
   ```

2. **تحقق من موقع الصورة:**
   - يجب أن تكون الصور في مجلدات فرعية: `plugins/CraftMenu/images/folder/image.png`
   - وليس في الجذر: `plugins/CraftMenu/images/image.png`

3. **تحقق من صيغة الصورة:**
   - فقط ملفات PNG مدعومة
   - تأكد من الامتداد الصحيح (`.png`، وليس `.PNG`)

4. **تحقق من تحميل حزمة الموارد:**
   - يجب تكوين حزمة موارد الخادم
   - يجب أن يقبل اللاعب حزمة الموارد

5. **أعد تحميل الإضافة:**
   ```
   /cm iadat
   ```

## القائمة لا تفتح

**الأعراض:** أمر `/cm iftah` لا يفعل شيئًا.

**الحلول:**

1. **تحقق من وجود القائمة:**
   ```
   /cm qaima
   ```

2. **تحقق من وحدة التحكم للأخطاء** بعد تشغيل الأمر

3. **تحقق من بناء YAML:**
   - استخدم مدقق YAML
   - تحقق من المسافات البادئة الصحيحة

4. **تأكد من صحة موقع الظهور:**
   - يجب أن يكون العالم محملاً
   - يجب أن يكون الموقع قابلاً للوصول

## التصادم لا يعمل

**الأعراض:** المؤشر لا يكتشف الويدجت.

**الحلول:**

1. **تفعيل جسيمات التصحيح:**
   ```
   /debugcollision tabdil
   ```

2. **تحقق من تكوين التصادم:**
   ```yaml
   collision:
     enabled: true
     size: {x: 0.1, y: 0.1, z: 0.1}
   ```

3. **زد حجم صندوق التصادم** إذا كان صغيرًا جدًا

4. **تحقق من موضع الويدجت** - قد يكون التصادم منزاحًا

## الأصوات لا تعمل

**الأعراض:** إجراءات الصوت ليس لها تأثير.

**الحلول:**

1. **للأصوات المخصصة:**
   - ضع ملفات `.ogg` في `plugins/CraftMenu/sounds/folder/`
   - أعد توليد حزمة الموارد: `/cm hazma`

2. **لأصوات Minecraft:**
   - استخدم الصيغة الصحيحة: `minecraft:ui.button.click`

3. **تحقق من إعدادات مستوى الصوت** في تكوين الإجراء

## مشاكل الأداء

**الأعراض:** تأخر عند استخدام القوائم.

**الحلول:**

1. **حسّن الصور:**
   ```
   /cm suwar maseh
   /cm suwar islah --backup
   ```

2. **قلل تكرار الرسوم المتحركة** في القوائم المعقدة

3. **عطّل وضع التصحيح:**
   ```yaml
   craftmenu:
     general:
       debug: false
   ```

4. **زد فترة التحديث:**
   ```yaml
   craftmenu:
     performance:
       update-interval: 2
   ```

## الإضافة لا تتحمل

**الأعراض:** الإضافة تعرض أخطاء عند البدء.

**الحلول:**

1. **تحقق من إصدار Java:**
   - يتطلب Java 17 أو أعلى

2. **تحقق من التبعيات:**
   - يجب تثبيت PacketEvents

3. **تحقق من إصدار الخادم:**
   - يتطلب Minecraft 1.20.4+

4. **راجع سجلات البدء** للأخطاء المحددة

5. **جرب الاسترداد:**
   ```
   /cm istirja
   ```

## أخطاء YAML

**الأعراض:** الأخطاء تذكر تحليل YAML.

**المشاكل الشائعة:**

1. **مسافات بادئة غير صحيحة:**
   ```yaml
   # خطأ
   widgets:
   my_widget:
     type: IMAGE

   # صحيح
   widgets:
     my_widget:
       type: IMAGE
   ```

2. **علامات اقتباس مفقودة حول القيم الخاصة:**
   ```yaml
   # خطأ - & له معنى خاص
   title: &bHello

   # صحيح
   title: "&bHello"
   ```

3. **صيغة قائمة غير صحيحة:**
   ```yaml
   # خطأ
   events:
     on_cursor_click:
       action: sound

   # صحيح
   events:
     on_cursor_click:
       - action: sound
   ```

## الحصول على المساعدة

إذا كنت لا تزال تواجه مشاكل:

1. تفعيل وضع التصحيح وتحقق من مخرجات وحدة التحكم
2. تحقق من مشاكل GitHub للمشاكل المعروفة
3. أنشئ مشكلة جديدة تتضمن:
   - إصدار الخادم
   - إصدار الإضافة
   - سجلات وحدة التحكم
   - ملفات التكوين (أزل البيانات الحساسة)
